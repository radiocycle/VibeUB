# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

- Initial public VibeUB release.
- Added modular userbot core with built-in command loader.
- Added inline bot support with BotFather automation and inline runtime.
- Added inline config panel with module categories, option screens, typed values, and reset-to-default actions.
- Added owner-only inline access control.
- Added language packs and language selection menu for English and Russian.
- Added configurable `info` and `ping` modules with placeholders, banner URL support, quote media, and invert media options.
- Added dynamic module management, aliases, logs, restart, eval, and terminal commands.
- Added help improvements, core/external grouping, and fuzzy module lookup.
- Added module authoring documentation and updated installation/troubleshooting docs.
